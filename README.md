## About the Project
자칭 빵 맛집 전문가들은 뚜렷한 인증제도가 없어서 쉽사리 무시되곤 한다. 빵을 사랑하고 세상의 모든 빵 맛집을 꿰뚫고 있는 그들에게 빵 전문가 자격증을 발급해줌으로서 자존감을 회복할 수 있는 계기를 마련하고자 했다.
